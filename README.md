# Psylocke

This repository contains the authentication and general backend for Todd.

## Quick Start

Prerequisites:

- Node.js: 18.0.0 or higher
- npm: 8.0.0 or higher

```bash
npm i
npm run dev
```

## Specifications/Tooling

- Next.js as an API framework
- PostgreSQL + Drizzle ORM
- Vitest for testing
